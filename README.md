## Getting Started

This repository contains all the samples files for Urovo Android Smart POS Payment Terminal;Handheld Data Terminal.

- Open the project in Android Studio.
- Compile and deploy to your Urovo Android device (such as a Smart POS I9000S/I9100 model, PDA I6310/DT50/DT40/DT30 model).
- Need more information about getting started with Urovo Android device? Check the [SDK_ReleaseforAndroid](https://github.com/urovosamples/SDK_ReleaseforAndroid)
